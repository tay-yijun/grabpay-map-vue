# GrabPay Map Vue

## Project Description

This project was created using [Vue CLI](https://github.com/vuejs/vue-cli). 

##### Technologies

* Vue.js
* mapbox-gl
* lodash
* bootstrap-vue
* axios 

## Setup

##### Install dependencies
```
npm install
```
##### Compiles and hot-reloads for development
```
npm run serve
```
##### Compiles and minifies for production
```
npm run build
```
##### Lints and fixes files
```
npm run lint
```
